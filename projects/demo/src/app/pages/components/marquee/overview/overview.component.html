Marquee

The @elementar/components/marquee library is an Angular package that delivers a versatile marquee component, allowing for customizable scrolling content either horizontally or vertically. It offers various configuration options, including animation speed, direction (horizontal or vertical), reverse scrolling, gap size, and the ability to pause on hover.

Features

  • A scrolling marquee with customizable options for both horizontal and vertical movement.
  • Includes options to control animation speed, direction, and reverse scrolling.
  • Allows the marquee to pause when hovered.
  • Lightweight and simple to integrate as a standalone component.
